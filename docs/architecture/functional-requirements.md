# Functional Requirements

## User Management

Users should be able to:

- Register
- Login
- Update profile
- Add address

## Provider Management

Service providers should be able to:

- Register
- Add skills
- Set availability
- Accept or reject bookings

## Search

Users should be able to:

- Search carpenters near their location
- View provider details

## Booking

Users should be able to:

- Create booking
- Cancel booking
- View booking status

Providers should be able to:

- Accept booking
- Complete booking

## Review System

Users should be able to:

- Rate providers
- Leave feedback
