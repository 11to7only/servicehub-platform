# ServiceHub Platform - System Overview

## Introduction

ServiceHub is a microservices-based platform that allows users
to find carpenters near their location and book services.

The platform connects service providers (carpenters) with users
who require home repair services.

## Objectives

The objective of this project is to design and implement
a scalable backend platform using modern cloud technologies.

The project will demonstrate the use of:

- Microservices architecture
- Event driven communication
- Containerization
- CI/CD pipelines
- Monitoring and logging
